Trello

Trello is a versatile application that can be used to manage large projects or even your grocery shopping list.

  1. Markdown
  3. Markdown editors

Overview

Trello is an extremely popular Kanban-style task management program.

One of the tool's strong points is the possibility of dragging and dropping the cards inside the interface, facilitating the reorganization of the project.

First you create tables and add columns, which app calls lists, and inside these columns insert cards that will contain your tasks.

Support for Markdown in Trello covers much of the basic syntax of the language. You can use Markdown in card comments, checklists and the author's bio.

It is worth noting that when using iOS or Android devices, some Markdown formatting is not displayed as you expect.

More information about Trello on the official website https://trello.com/.
