StackEdit

StackEdit is a robust Markdown editor that runs directly in your browser.

  1. Markdown
  3. Markdown editors
  5. StackEdit

Overview

StackEdit is an extremely versatile editor, it offers the user several features and configuration options.

Despite being a Web tool, StackEdit can compete with various Desktop applications, often presenting itself as a better option.

The program exports your documents to various file formats, supports synchronization using third-party services and you can also customize templates using metadata.

The syntax support of Markdown is incredible, the tool is even compatible with LaTeX and UML diagramming.

You can also use StackEdit without being connected, even if there is no connection to internet, the program will still load in your browser, of course, synchronizing documents will not be possible due to the lack of connection.

The app documentation is one of the project's weaknesses. Most of the time you will have to learn the features of the tool using trial and error.

A clear example of the problem of lack of documentation is the creation of templates. For example, you can define your own templates for exporting files, but finding information on how to do this is another story.

This situation can lead novice users to never use most of StackEdit's features, simply because they don't even know they exist.

You can even use the community support forum, but, let's face it, it would be much more practical if there were instructions directly in the application.

The official website of StackEdit is located at https://stackedit.io/.

Unordered lists Yes

Horizontal lines Yes
Links Yes
Images Yes StackEdit does not support sending images directly on the tool's website, it is necessary to store them on another server and use only the links of the images to link them to your Markdown document.
Tables Yes
Fenced code blocks Yes StackEdit is also compatible with highlighting the syntax of the content within the block.
Footnotes Yes
Title Id Partial Title identification is automatically generated by app, it is not possible for the user to specify custom ids.

Strikethrough Yes
Task lists Yes
Emoji (copy and paste) Yes
