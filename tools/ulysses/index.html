Ulysses

Ulysses is one of the most acclaimed applications by writers who need professional resources for creating documents on Mac, iPad and iPhone devices.

  1. Markdown
  3. Markdown editors
  5. Ulysses

Overview

The number one application for writers and reviewers who use Apple devices has theme and file export options that are difficult to find in other programs.

However, when we talk about Markdown, the tool leaves a lot to be desired.

Support for Markdown in Ulysses is restricted to a subset of the language syntax, but even so, some elements considered as basic are not present.

Another, at least peculiar, factor is the use of non-standard Markdown notations to create elements of ... Markdown?

For these reasons, using Ulysses to edit Markdown documents can become a frustrating task, if you are just starting out or don't want to learn a new language syntax, it is better to use another program.

The official website for Ulysses is at https: //ulysses.app/.

Support for Markdown on Ulysses

The Table 1.1 shows the elements Markdown compatible with Ulysses, pay attention to the observations of some elements to understand how to build them using the alternative syntax of the tool.

Element Support Information
Titles Yes Ulysses does not provide compatibility with alternative syntax for creating titles.
Paragraphs Yes
Line breaks No
Bold Yes
Italic Yes
Quote block Yes
Ordered lists Yes
Unordered lists Yes
Code Partial code blocks are not supported by the tool.
Horizontal lines Partial To form a horizontal line in Ulysses it is necessary to use four dashes: ----.
Links No Ulysses does not support the Markdown syntax for entering links, but you can use the Markup menu to perform the same task.
Images No The program is not compatible with the Markdown syntax, use the menu Markup to insert images in your document.
Tables No
Fenced code blocks No Use the Markup menu to insert blocks of code in Ulysses, Markdown syntax is not supported.
Footnotes No Similar to several elements, this must also be inserted using the Markup menu, the Markdown syntax for footnotes, unfortunately, is not supported by app.
Title Id No
Definition lists No
Strikethrough Partial To strike text in Ulysses use two vertical bars around the content, for example: ||strikethrough text||. The standard Markdown syntax is not supported.
Task lists Yes The creation of task lists in Ulysses can be done using only the brackets [], without the need to add a dash - at the beginning of the line.
Emoji (copy and paste) Unknown
Emoji (access codes) Unknown
automatic URL Link No
Disable automatic link from URL Yes It is not necessary as Ulysses does not automatically link links.
HTML No
